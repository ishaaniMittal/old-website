Home Automation System

Home Automation System which lets the user enter their homes using facial recognition

Prototyping
Image Processing
Prototyping
Image Processing

Motivation

The project was made as a part of the minor project during the junior year. The main motivation behind the project was to provide a cheap way to automate and regulate the home entry in the house. It removes all dependency on keys and headache to carry keys all the time. The project also provides a level of security as only a set of fed faces would be granted access.

Device Overview

The device was prototyped using Raspberry Pi attached to LED screen and a motor. The first part contains enrolling the face into the system. A mean image and an Eigen image is formed from these images using Eigen faces algorithm. At the time of login, the captured image is projected on this eigen face obtained using Principal Component Analysis in order to carry out face recognition. The output of the software is then displayed on the LED screen. If successful, the motor attached to the door knob is turned to open the door.

Login Attempts


In the beginning, a few images of the person at different angles and poses was fed to the system. While login, different poses were checked on. The image taken for login is at first passed through the face identification module, which helps extract the face from the image. After the face has been extracted, face recognition algorithm is run to match the login attempt image with the training set image. An analysis of the above login attempts with the training set gave an accuracy of 100%. The system does not work well with varying backgound and lighting.
